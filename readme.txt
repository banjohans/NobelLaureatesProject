Hoy @Fullstack  Her er oppgaveteksten for de som mangler tilgang til Canvas: 

Velg et datasett i JSON format fra dette Github repoet: https://github.com/jdorfman/awesome-json-datasets

Lag en folder Models, hvor alle modellene dine skal eksistere. 
Lag en model.cs som matcher JSON filen, slik at den kan deserialiseres, hvis JSON filen ikke er lett å querye, kan det hende du må prosjektere denne JSON modellen til en ny modelltype. 
Lag en Context mappe som har en context.cs fil, som holder jsondataen din i minnet.
Lag noen spørringer mot datasettet, output av spørringene skal lagres i JSON format. Minst en spørring pr følgende metode:
Select(),
SelectMany(),
GroupBy(),
   BONUS:

Lag en input som lar en bruker søke på et eller flere parametere selv. 

 

Innlevering: Lenke til Github repository
